## 📌 Project Overview

The **Automatic Water Level Indicator and Controller** is an embedded system designed to monitor and control the water level in a storage tank using an Arduino UNO microcontroller and an ultrasonic sensor.

The system prevents:

- Water overflow  
- Dry running of the pump  
- Manual monitoring  

It provides real-time water level percentage display and automatic pump control.

---

## 🎯 Objectives

- Monitor water level using ultrasonic sensing
- Automatically control water pump
- Display real-time tank percentage
- Provide Manual and Auto mode selection
- Store tank height using EEPROM
- Reduce water and electricity wastage

---

## 🛠 Hardware Components

- Arduino UNO (ATmega328P)
- HC-SR04 Ultrasonic Sensor
- 16x2 LCD Display (I2C)
- 5V Relay Module
- 12V DC Water Pump
- Push Buttons (SET & MODE)
- Jumper Wires
- Breadboard / PCB
- Power Supply (5V / 12V)
